## Overview

This project focuses on analyzing customer churn behavior in the banking sector using data analytics and business intelligence techniques.

The project was developed as part of the Unified Mentor internship program and aims to identify customer retention patterns, churn indicators, and business insights that support strategic decision-making in the banking industry.

The analysis explores customer demographics, account behavior, and churn-related trends to better understand factors contributing to customer attrition.

## Business Problem

Customer churn is a major challenge in the banking industry because losing customers directly impacts profitability and long-term customer value.

Banks need to identify high-risk customer segments and understand churn behavior to improve retention strategies and customer satisfaction.

This project analyzes customer churn data to uncover behavioral patterns and generate actionable business insights.

## Internship Context

This project was completed as part of the Unified Mentor internship program with a focus on practical business analytics, customer churn analysis, and data visualization.

## Key Business Questions

* Which customer groups have the highest churn rates?
* How does customer tenure affect churn behavior?
* What demographic patterns exist among churned customers?
* Which financial indicators influence customer attrition?
* How can churn insights improve retention strategies?


```bash
streamlit run app/app.py
```
